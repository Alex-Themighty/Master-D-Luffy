Theory: The Comprehensive Formulas

Rule ID Standard Rule $(x)$ Chain Rule $(u)$
Rule 1: Power$(x^\alpha)' = \alpha x^{\alpha-1}$$(u^\alpha)' = \alpha u^{\alpha-1} \cdot u'$
Rule 2: Sqrt$(\sqrt{x})' = \frac{1}{2\sqrt{x}}$$(\sqrt{u})' = \frac{u'}{2\sqrt{u}}$
Rule 3: Inverse$(\frac{1}{x})' = -\frac{1}{x^2}$$(\frac{1}{u})' = -\frac{u'}{u^2}$
Rule 4: Sine$(\sin x)' = \cos x$$(\sin u)' = u' \cdot \cos u$
Rule 5: Cosine$(\cos x)' = -\sin x$$(\cos u)' = -u' \cdot \sin u$
Rule 6: Tangent$(\tan x)' = \frac{1}{\cos^2 x}$$(\tan u)' = \frac{u'}{\cos^2 u}$
Rule 7: Cotangent$(\cot x)' = -\frac{1}{\sin^2 x}$$(\cot u)' = -\frac{u'}{\sin^2 u}$
Rule 8: Exp (e)$(e^x)' = e^x$$(e^u)' = u' \cdot e^u$
Rule 9: Exp (a)$(a^x)' = a^x \ln a$$(a^u)' = u' \cdot a^u \ln a$
Rule 10: Ln$(\ln x)' = \frac{1}{x}$$(\ln u)' = \frac{u'}{u}$
Rule 11: Log (a)$(\log_a x)' = \frac{1}{x \ln a}$$(\log_a u)' = \frac{u'}{u \ln a}$
Rule 12: Product$(fg)' = f'g + fg'$
Rule 13: Quotient$\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
Rule 14: Logarithmic Diff$(u^v)' = u^v \left[ v' \ln(u) + v \frac{u'}{u} \right]$
